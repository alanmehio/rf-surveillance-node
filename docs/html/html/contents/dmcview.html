Module contents

DMC View, a Python Package Desktop application to view the data from dmc device
Goal of this project is to allow device manufacturer to plug the device from a serial port and view the Digital Magnetic Data. Each device manufacturer has its own unique serial port Id; so there is no need to configure. It is just plug and play approach with different internal adapter
